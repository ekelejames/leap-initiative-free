# 🚀 LEAP Initiative - DevOps/Kafka/Fullstack Practice Playground

The LEAP Initiative (Learn, Execute, Automate, Perfect) is a DevOps/Kafka/Fullstack practice playground designed to help you spin up and manage various services for hands-on learning and experimentation.


## Getting Started
### Prerequisites

- Docker Desktop installed and running
- Docker Compose

### Installation

1. Clone or download this repository
```bash
git clone https://github.com/ekelejames/leap-initiative-free.git
```
2. Navigate to the project directory 
```bash
cd leap-initiative-free
```
3. Run the following command:
```bash
docker-compose up -d
```

### Access the Application
Once the containers are running, access the LEAP Initiative playground in your browser at:
```bash
http://localhost:3001
```

## Stopping the Application

To stop all running containers:

```bash
docker-compose down
```
